# SwipeStudy

This project is the completed SwiftUI version of the SwipeStudy application for Module 7 of the Mobile Application Development course.

SwipeStudy is a simple flashcard study app focused on gestures, animations, and interactive mobile UI behavior using SwiftUI.

The project includes:

* Basic SwiftUI project structure
* Flashcard model
* Sample flashcard data
* Main flashcard study view
* Reusable stats header view
* Reusable flashcard view
* Local session state
* Got It, Review, and Favorites counters
* Tap gesture to flip between question and answer
* Drag gesture to move the card
* Swipe right action for Got It
* Swipe left action for Review
* Long press gesture to mark a card as favorite
* Card movement animation
* Card flip animation
* Swipe labels for Got It and Review
* Reset Session feature
* Completion view for the end of the deck

This app does not use login, Firebase, local storage, Core Data, or a database. The flashcards are hard-coded, and the counters only exist during the current app session.

Open the project in Xcode and run the application on the iOS Simulator or a physical iOS device.
